# AWS S3 Static Website Hosting

## Project Overview
This project demonstrates hosting a static website using Amazon S3.

## Steps Performed
- Created an S3 bucket
- Configured public access using bucket policy
- Enabled static website hosting
- Uploaded HTML files
- Enabled versioning for backup
- Verified website using S3 endpoint

## Services Used
- Amazon S3
- AWS IAM

## Screenshots
Screenshots are included to show:
- S3 bucket creation
- Website hosting configuration
- Successful website access

## Outcome
Successfully hosted a static website using AWS S3 with proper security and versioning.
